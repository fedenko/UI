A) Basic Installation - top

Glass menu is a pure css based menu that can support all your needs, installation is easy as pie. Lets get started !!

  • First import the Glass.css file in the html file
    •        
      <link rel="stylesheet" type="text/css" href="assets/Glass.css"  />
  • Now just create a menu as you usually do adding list items to an unordered list with each li item has an anchor tag.
  • Finally add the class 'GlassMenu' . You are done.
       
     <ul class="GlassMenu">
	<li><a href="#">Home</a></li>
	<li><a href="#">Services</a></li>
    <li><a href="#">Portfolio</a></li>
    <li><a href="#">About Us</a></li>
    </ul>

converts to -

  • Home
  • Services
  • Portfolio
  • About Us

Creating sub menus

Now creating sub menus is easy. Just like you have create a menu, you can create a sub menu. In Glassmenu all the subelements are wrapped inside a div with class 'subglass'. So to create a submenu create an unordered list with your submene items and wrap in inside the div with subglass. Then insert it in the li of the main main after the anchor tag. Note sub menu list does not have any classes in them. 

       
     <ul class="GlassMenu">
	<li><a href="#">Home</a></li>
	<li><a href="#">Services</a> <!-- This one will have submenu  -->
    
    <div class='subglass'>
     <ul>
	<li><a href="#">Web design</a></li>
	<li><a href="#">iPad</a></li>
    <li><a href="#">SEO</a></li>
    <li><a href="#">Software</a></li>
    </ul>
    </div>
    
    </li>
    <li><a href="#">Portfolio</a></li>
    <li><a href="#">About Us</a></li>
    </ul>

Working with forms.

Working with forms is same as adding list, just create a div with class 'subglass' and add your form thats it !!

       
        <ul class="GlassMenu">
    <li><a href="#">Shopping List</a>
     <div class="subglass">
      
      <form>
     <fieldset>
     <label for="user">Username</label>
     <input type="text" id="user" />
     <label for="pass">Password</label>
     
     <input type="password" id="pass" />
     <input type="submit"  value="Login" />
     </fieldset>
     
     </form>
       
      </div>
     
     </li>
    </ul>

The above code gives you -

  • Login

Same way you can add divs or anything you like. Just make sure they are wrapped inside subglass div.

CSS Classes details -

This sections covers all you need to know about the different classes and styling for your own custom theme purposes.

  1. .Glassmenu is the parent container and is the first class available to you. It main element in which you can defined font size, type etc. 
    2.       
.GlassMenu {  zoom:1; background:#1c1c1c; font-family: Tahoma, Geneva, sans-serif; 
 font-weight:500; font-size:16px; padding:8px 5px;  display:block;  list-style:none;   }
.GlassMenu:after { content:"."; display:block; visibility:hidden; height:0; clear:both;  }
  2. Then we have basic styling for the li and anchors of the main menu items. That is the top level menu items. Position relative is important for li elements, from this the sub menu places itself.
    3.        
.GlassMenu  li { float:left; padding:10px 15px ; position:relative;   }
.GlassMenu a  { text-decoration:none; color:#fff; display:block; }
  3. Next comes the styling of the the sub menu items. If you want to style the submenu items below are the classes who want to change. Here you should note that the width of the sub menus if decided in the .GlassMenu li ul. You can change if here. One more thing to remember is that if you change the width of the sub menu, do change the value of the left attribute in the .GlassMenu .subglass div class. This class alignes the sub sub menus left. The value should be equal to the width you have given. 
    4.        
 .GlassMenu  li ul{  width:170px; list-style:none; padding:3px; margin:0px; background:#1c1c1c; border:1px solid #373737; 
 -moz-border-radius: 3px; -webkit-border-radius: 3px; font-size:14px; position:relative;  }
.GlassMenu  li ul li{  float:none; color:#fff; border-bottom:1px solid #292929;border-top:1px solid #121212; padding:10px;margin:0px; }
.GlassMenu  li ul li:first-child{  border-top:none;  }
  4. Now comes the subglass that we used to wrap our elements into. The basic styling allows to create a glass like border with image in the i folder. If you want to change the path that image you can do it here. Plus there is styling on sub sub menu items too. Finally there is styling for li on hover state.
    5.        
.GlassMenu .subglass { display:none; background:url(i/glass.png); padding:7px; margin-top:11px; -moz-border-radius: 5px; 
-webkit-border-radius: 5px; position:absolute;  z-index:40; left:0; }
.GlassMenu .subglass div{ margin-top:-20px; z-index:41; left:170px;  }
.GlassMenu li:hover>div { display:block;}
  5. Next module deals with the styling on generic html elements that are wrapped inside a subglass like divs, images, p ,lists etc. Note the first class .GlassMenu .subglass>div is the one, which if the html elements overflow you might want to change the width. It was mentioned earlier too in adding shopping list section. 
    6. 
.GlassMenu .subglass>div { margin:0px;  background:#1c1c1c; padding:7px;  -moz-border-radius: 5px; -webkit-border-radius: 5px; float:left;  width:450px;  }
.GlassMenu div.subglass>div h2,h4 { font-family: "Lucida Sans Unicode", "Lucida Grande", sans-serif; font-weight:500; 
font-size:20px; padding:3px;border-bottom:1px solid #333; margin-bottom:4px; }
.GlassMenu div.subglass>div h4 {  font-weight:200; font-size:16px; }
.GlassMenu .subglass>div img { width:400px; height:250px;}
.GlassMenu .subglass>div ul {  float:left; border:none; width:140px; }
.GlassMenu .subglass>div li,p { border-bottom:1px dotted #333; border-top:none; font-size:12px; display:block;  }
.GlassMenu .subglass>div li:first-child { border-top:1px dotted #333;}
  6. Finally there is basic styling for form elements. You can easily style them yourself.
    7.       
 .GlassMenu form { background:#fff; -moz-border-radius: 2px; -webkit-border-radius: 2px; padding:20px; margin:5px; width:500px; 
 font-family:"Lucida Sans Unicode", "Lucida Grande", sans-serif  }
.GlassMenu input[type=text],input[type=password],textarea {border:none; background:#ebebeb; outline:1px solid #ddd; 
border-left:1px solid #ccc;border-top:1px solid #a7a7a7;
 padding:5px;  }
 .GlassMenu  legend { padding:3px; margin-left:30px; color:#0098ef;}
.GlassMenu  fieldset { border:#ccc 1px solid; padding:30px; }
.GlassMenu label,input[type=text],input[type=password],textarea { width:100%;margin:4px 2px; color:#000;font-family:Tahoma, Geneva, sans-serif;  }
.GlassMenu input[type=submit] { background:#c4b1a0; margin:2px; color:#FFF; outline:none; border:1px solid #dbc5b1; 
padding:6px 10px; font-family:Tahoma, Geneva, sans-serif; font-weight:800; }
.GlassMenu input[type=submit]:hover {border:1px solid #fff;  }
.GlassMenu input[type=text]:focus,input[type=password]:focus,textarea:focus  { border:1px solid #c4b1a0; }

Making Menu stick

Making the menu is very easy , just add the 'sticky' class in the main menu ul's class. That's it !

       
     <ul class="GlassMenu sticky">
	<li><a href="#">Home</a></li>
	<li><a href="#">Services</a></li>
    <li><a href="#">Portfolio</a></li>
    <li><a href="#">About Us</a></li>
    </ul>
