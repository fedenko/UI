Installation

Installing Google Content Locker could not be any easier!

  1. Upload the google-locker folder to your WordPress plugin directory ( /wp-admin/plugins/ )
  2. Activate the plugin from your WordPress plugins administration page.
  3. You are ready to rock! Go to your post / page editor to see your shiny new MCE toolbar button.

You can change the localization by editing the plugin files first option to include the locale you want to use. You can find a list of them here: http://code.google.com/apis/+1button/#languages

How to Use

Basic Usage Information

Using the Google Content Locker is just as easy as installing it! When you are using the post / page editor select some content you want to lock and then click the Google Content Locker icon in your mce toolbar. (You must be in rich editor mode) and you will get a pop up allowing you to configure your content locker.

You can specify which color theme the locker will have, the URL to be shared on Google when the +1 button is clicked, and the message that should appear inside the content locker.

Note that Google requires you use a public, valid URL with the +1 button in order for it to work. (Must return a header code of 200 OK)

 

Multiple Lockers on the Same Page

Although you can use any number of content locker in a post / page as you would like to there are some things to keep in mind and limitations set in place by Google.

Because a URL can only be +1'ed one time, you can only use a URL once in each post / page. You can use the same URLs however many times you want in different posts but for each post you must use different URLs for each locker within the post / page. So if you have 3 content lockers in one post, those 3 lockers will need to have different links but you can use those links in other posts and pages as well.

Currently Google does not support checking if a user already has +1'ed a given URL.
